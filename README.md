A simple image matching game focused on UI design using Flutter.

<Similarity>
Similarity is a visually engaging memory game built with Flutter, where players match two identical images. The project emphasizes clean UI design, smooth animations, and immersive sound effects to provide an intuitive gaming experience.

<Overview>
The game features multiple levels of increasing difficulty. Each level challenges the player’s memory and observation skills. The project showcases Flutter layout management, asset integration (images & audio), and responsive UI design.

<Features>
-Multiple levels with increasing difficulty
-Clean and responsive UI design
-Background music and sound effects for interactions
-Settings screen to control audio and preferences
-Home screen for easy navigation
-Match-pair game mechanics
